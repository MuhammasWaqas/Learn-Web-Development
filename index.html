What is HTML5?

HTML5 (Hypertext Markup Language 5) is a markup language used for structuring and presenting hypertext documents on the World Wide Web. HTML5 is an easy-to-learn, free, popular, and constantly evolving language that runs on any device. It's constantly updated with new modules and feature releases. It is a language you're bound to encounter in the future, so it makes sense to at least learn the basics.

What is CSS3?

CSS3, also known as Cascading Style Sheets. CSS3 is used for the same thing as CSS, namely to style web pages and make them more attractive and user-friendly. CSS is used to style and layout web pages, e.g to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

What is Bootstrap5?

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.Bootstrap is designed to be easy to use all you need is a basic understanding of web layouts and front end coding languages like HTML and CSS.

What is a JavaScript Framework?

A JavaScript framework is a pre-written collection of JavaScript code that provides structure and tools to simplify and speed up web application development.
